# 🌸 SakhiCare – Women Healthcare Platform

SakhiCare is a comprehensive digital healthcare platform designed to empower women by providing accessible, reliable, and personalized health information across different stages of life — from menstrual health to pregnancy and baby care.

---

## 🎯 Project Objective

The goal of SakhiCare is to bridge the gap between women and essential healthcare resources by offering a centralized platform for:

* Health awareness
* Preventive care
* Maternal support
* Infant care guidance

---

## ✨ Key Features

### 👩‍⚕️ 1. Women Health Management

* Menstrual cycle tracking system
* Menstrual health awareness and education
* Nutritional guidance during menstrual cycles
* Myth-busting section for common misconceptions

---

### 🤰 2. Pregnancy Care System

* Complete pregnancy guidance from early stages to delivery
* Dos and Don’ts during pregnancy
* Emotional and mental health support
* Yoga and fitness guidance for pregnant women
* Post-pregnancy recovery information

---

### 👶 3. Baby Care Module

* Newborn care guidelines
* Baby hygiene and comfort care
* Vaccination schedules
* Common baby health issues and solutions
  
---

### 🛍️ 4. Product Awareness & Recommendations

* Guidance on essential baby and women care products
* Usage instructions for products like diapers and pads
* Skin care and hygiene product awareness

---

### 📅 5. Appointment Booking System

* Users can book healthcare appointments
* Post-booking confirmation page

---

### 🔐 6. User Authentication System

* User registration and login functionality
* Secure access to personalized features

---

### 📧 7. Communication & Support

* Contact form for user queries
* Email response/confirmation page

---

### 🌐 8. General Website Pages

* Homepage with platform overview
* About page explaining mission and vision
* Privacy policy and terms of use

---

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/Vaishnavi688/SakhiCare.git
```

2. Navigate to the project folder:

```
cd SakhiCare
```

3. Run the server:

```
python manage.py runserver
```

4. Open in browser:

```
http://127.0.0.1:8000/
```
